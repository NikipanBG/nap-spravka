☁ Google Drive — настройка

За да избираш снимки директно от Google Drive на компютъра, въведи свои Google идентификатори (еднократно, пазят се само в браузъра).

📱 На телефон не е нужна настройка! Натисни „📁 Файл" и системният избор вече показва Google Drive, Снимки и Камера.
Как да ги взема? (стъпки)
  1. Отвори console.cloud.google.com и създай проект.
  2. APIs & Services → Library → включи Google Drive API и Google Picker API.
  3. Credentials → Create credentials → API key → копирай в „API Key".
  4. Credentials → Create credentials → OAuth client ID → тип Web application.
  5. В „Authorized JavaScript origins" добави адреса на сайта (напр. https://потребител.github.io).
  6. Копирай Client ID в полето горе.
